# velleb-multijob
This is a multi-job system for QB-Core framework allowing players to have multiple jobs and switch between them from an NPC context menu.

## Dependencies
- QB-Core (More frameworks may be added in the future)
- https://github.com/communityox/ox_lib
- https://github.com/communityox/oxmysql

## Installation Guide
1. Run the qb.sql file to add the player_jobs table
2. Configure script in `shared/config.lua`
